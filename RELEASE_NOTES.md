# Release v0.1.0

Initial MoonByte VM release.

## Added

- Hand-written lexer and parser for a compact Lua-like language.
- AST model and bytecode compiler.
- Stack-based virtual machine with globals, local slots and function calls.
- Table literal support with property reads.
- Host function embedding API.
- CLI commands: `run`, `disasm`, `bench`, `repl`, `version`.
- Self-contained test runner without external test packages.
- Mini game-loop embedding example.
- Architecture and language documentation.
- GitHub Actions CI.
